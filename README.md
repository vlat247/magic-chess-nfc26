**Arcane Chess** — это полноценная игровая Web-платформа в стиле тёмного ретро-фэнтези, сочетающая классические шахматы с элементами ролевых игр (RPG), магическими заклинаниями, продвинутым искусственным интеллектом, многопользовательским режимом в реальном времени и продуманными бизнес-механиками. 

Проект разработан специально для программы **nfactorial** и воплощает в себе уровень **«Великий» (Level Great)** — готовый к запуску прототип стартапа с высокой вовлеченностью, надежной архитектурой безопасности и механизмами монетизации.

---

## 🌟 Ключевые возможности (Features)

*   **🔮 Режим «Chaos Spell Mode»:** Уникальный игровой процесс, где фигуры могут использовать разрушительные заклинания (*Glacial Shatter*, *Void Vortex*, *Solar Flare*) с кастомными анимациями и визуальными эффектами.
*   **🤝 Мультиплеер в реальном времени:** Создание закрытых комнат и игра с другом по прямой ссылке, реализованная на базе Supabase Realtime (WebSockets) с защитой от десинхронизации и умным отслеживанием дисконнектов.
*   **🤖 Двухдвижковый ИИ-Тренер (AI Coach):** Послематчевый анализ партий с расчетом точности ходов (Accuracy), среднего рейтинга потери пешек (ACPL) и тактическими подсказками от «Архимага Сетки».
*   **🏆 Система «City Guilds» (Лидерборды городов):** Соревновательная система с глобальным топом и возможностью отслеживать лучших игроков по конкретным городам (например, Алматы, Астана), их винстрикам и уровням.
*   **🛡️ Система Прогрессии и Достижений:** Начисление опыта (XP), прокачка уровней и автоматическое разблокирование достижений (*First Blood*, *Spellbinder*, *Unstoppable*, *Grandmaster*) прямо во время завершения партий.
*   **💎 Магазин косметики и Pro-Абонемент:** Premium-раздел со сравнением тарифов, сезонным пропуском и реалистичным симулятором транзакций (Stripe Simulation) для покупки уникальных скинов фигур, досок и спецэффектов.

## 🚀 Выдающиеся технические решения (Our Engineering Pride)

### 1. 🧠 Двухуровневая ИИ-Архитектура (Stockfish + Qwen 32B via Hack Club API)
Система послематчевого анализа разделена на две части для максимальной производительности и минимизации задержек:
*   **Клиентская часть (Engine):** В песочнице Web Worker запускается скомпилированный под Emscripten шахматный движок **Stockfish 12** (`public/stockfish.js`). Он выполняет последовательный расчет партий до 12 полуходов, выявляет зевки (Blunders), неточности (Inaccuracies) и высчитывает Centipawn Loss (CPL) для каждого хода.
*   **Серверная часть (LLM Coach):** Эндпоинт `/api/coach` фильтрует наиболее критические ходы и отправляет структурированный запрос к мощной языковой модели **Qwen 32B (`qwen/qwen3-32b`)** через прокси-сервер **Hack Club AI API**. Модель анализирует контекст шахматной партии и возвращает строго типизированный JSON-пакет с текстовым разбором тактических ошибок на языке «Архимага Шахматной Сетки».

### 2. 🛡️ Авторитетная БД и строгая безопасность (Supabase RLS & PG Triggers)
Для предотвращения любых накруток опыта, ELO, достижений или обхода платных блокировок на клиенте, вся бизнес-логика перенесена на уровень СУБД PostgreSQL:
*   **Безопасная RLS-Политика:** Клиентам разрешено обновлять исключительно свои косметические настройки (выбранные скины) и публичные поля профиля. Изменение ELO, XP, уровня или массива купленных скинов строго запрещено:
    ```sql
    CREATE POLICY "Users can only update cosmetic and profile fields" 
      ON public.users FOR UPDATE USING (auth.uid() = id)
      WITH CHECK (
        auth.uid() = id AND 
        EXISTS (
          SELECT 1 FROM public.users 
          WHERE users.id = id AND users.rating = rating AND users.xp = xp AND users.level = level ...
        )
      );
    ```
*   **Автоматический процессинг на триггерах:** При добавлении записи в таблицу `matches`, триггер базы данных автоматически запускает PL/pgSQL функцию `process_match_stats()`. Она рассчитывает исход игры, начисляет ELO, опыт, проверяет условия достижений и выдает уникальные ачивки.

### 3. 🌐 Мультиплеер с защитой от десинхронизации и абандонов
*   **Анти-Десинхронизация:** Каждый ход инкрементирует монотонный счетчик `move_seq` в Supabase. Клиенты не примут ход с неверным порядковым номером, что гарантирует целостность партии.
*   **Graceful Disconnect:** Если игрок закрывает вкладку или теряет интернет-соединение, система переводит комнату в состояние ожидания и запускает таймер обратного отсчета для оставшегося игрока, позволяя ему досрочно забрать победу (TKO), если оппонент не вернется вовремя.

### 4. 💳 Реалистичный платежный шлюз (Stripe Simulator)
Модуль `/premium/checkout` предоставляет полноценную симуляцию оплаты:
*   Автоматическая группировка 16-значного номера карты пробелами по маске `#### #### #### ####`.
*   Форматирование даты окончания срока действия `MM/YY` с автоматической вставкой слеша.
*   CVV-маскирование и защита полей ввода.
*   Транзитные анимации загрузки транзакции (активация магического портала) и анимация успеха, начисляющая `pro` статус пользователю через безопасный Server Action.

## 🛠️ Стек технологий (Tech Stack)

*   **Core:** React 19, Next.js 16 (App Router), TypeScript.
*   **Database & Sync:** Supabase, PostgreSQL, Supabase Realtime (Channels/Realtime).
*   **State Manager:** Zustand (для управления комнатами и мультиплеерной логикой).
*   **Styling & FX:** Tailwind CSS 4.0, Framer Motion, Lucide Icons.
*   **Chess Logic:** `chess.js` (валидация ходов), `react-chessboard` (визуализация доски), Web Worker Stockfish.

## 🎮 Быстрый старт (Installation & Setup)

1.  **Клонируйте репозиторий:**
    ```bash
    git clone <repository_url>
    cd chess-second-try
    ```

2.  **Установите зависимости с помощью `pnpm`:**
    ```bash
    pnpm install
    ```

3.  **Настройте переменные окружения:**
    Создайте файл `.env.local` в корне проекта:
    ```env
    NEXT_PUBLIC_SUPABASE_URL=your_supabase_project_url
    NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
    AI_COACH_API_KEY=your_hackclub_api_key  # (По умолчанию используется встроенный ключ Hack Club)
    ```

4.  **Разверните базу данных:**
    Запустите SQL-скрипты `supabase.sql`, `multiplayer-schema.sql` и `monetization-progression-schema.sql` в вашем Supabase SQL Editor.

5.  **Запустите сервер разработки:**
    ```bash
    pnpm dev
    ```
    Откройте [http://localhost:3000](http://localhost:3000) в браузере.
